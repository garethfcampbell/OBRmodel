About the Model

This application allows you to explore the OBR's macroeconomic model. You can ask questions about the impact of various economic shocks and see how the model responds.

The model code and variable descriptions are pre-loaded. Simply type your question in the chat box to get started.

Welcome to the OBR Model Analyst. This tool is designed to assess the macroeconomic impact of fiscal policies, structural reforms, and external economic shocks using the Office for Budget Responsibility's macroeconomic model framework. Please describe the policy or scenario you wish to analyse, or select from the options below.
